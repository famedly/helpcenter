<div class="hero hero--primary">
  <div class="container">
    <h2 class="hero__title">Passwort zurücksetzen</h2>
    <p class="hero__subtitle">Wie Sie ihr Passwort zurücksetzen können, wenn Sie es vergessen haben oder ändern möchten.</p>
  </div>
</div>

:::info
Falls sich kein Browserfenster oder Tab für den Login öffnet, ist Ihr Account nicht auf einem TIM-Server und Sie müssen Ihren Administrator kontaktieren, um ein vergessenes Passwort zurückzusetzen.
:::

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>Öffnen Sie die Famedly Web App.</li>
    <li>Geben Sie die Anmelde-URL in das Eingabefeld ein.</li>
    <li>Klicken Sie auf <b>Weiter</b>.</li>
    <li>Ein neuer Tab zur Eingabebe der Login-Daten wird sich öffnen.</li>
    <li>Geben Sie Ihren Benutzernamen ein und klicken Sie auf **Weiter**.</li>
    <li>Klicken Sie auf <b>Passwort zurücksetzen</b>.</li>
    <li>Öffnen Sie Ihren E-Mail Posteingang und suchen Sie nach einer Mail mit dem Titel <b>Passwort zurücksetzen</b>. Sehen Sie gegebenenfalls auch im Spam Ordner nach.</li>
    <li>Klicken Sie auf den <b>Passwort zurücksetzen</b> Button in der Mail.</li>
    <li>Wählen Sie ein Passwort, das den Regeln unter dem Eingabefeld mit dem Titel <b>Neues Passwort</b> entspricht.</li>
    <li>Geben Sie Ihr neues Passwort in das Eingabefeld <b>Neues Passwort</b> ein.</li>
    <li>Geben Sie Ihr neues Passwort auch in das Eingabefeld <b>Passwort wiederholen</b> ein.</li>
    <li>Klicken Sie auf <b>Weiter</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Öffnen Sie die Famedly App.</li>
    <li>Geben Sie die Anmelde-URL in das Eingabefeld ein.</li>
    <li>Tippen Sie auf <b>Weiter</b>.</li>
    <li>Ein neuer Tab zur Eingabebe der Login-Daten wird sich öffnen.</li>
    <li>Geben Sie Ihren Benutzernamen ein und tippen Sie auf **Weiter**.</li>
    <li>Tippen Sie auf <b>Passwort zurücksetzen</b>.</li>
    <li>Öffnen Sie Ihren E-Mail Posteingang und suchen Sie nach einer Mail mit dem Titel <b>Passwort zurücksetzen</b>. Sehen Sie gegebenenfalls auch im Spam Ordner nach.</li>
    <li>Tippen Sie auf den <b>Passwort zurücksetzen</b> Button in der Mail.</li>
    <li>Wählen Sie ein Passwort, das den Regeln unter dem Eingabefeld mit dem Titel <b>Neues Passwort</b> entspricht.</li>
    <li>Geben Sie Ihr neues Passwort in das Eingabefeld <b>Neues Passwort</b> ein.</li>
    <li>Geben Sie Ihr neues Passwort auch in das Eingabefeld <b>Passwort wiederholen</b> ein.</li>
    <li>Tippen Sie auf <b>Weiter</b>.</li>
  </ol>
  </TabItem>
</Tabs>
