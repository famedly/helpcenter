## HBA Karte registrieren

Wenn Sie sich mit dem VZD Fhir Directory verbinden möchten, müssen Sie Ihren Heilberufeausweis (HBA) registrieren.
Dafür muss ein Mobilgerät verwendet werden, da eine NFC Verbindung benötigt wird, um die Karte zu lesen.
Nach der Registrierung werden Sie in der Lage sein, andere Nutzer und Kontaktpunkte anderer Organisationen
im VZD Fhir Directory zu finden. 

<Tabs>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf <b>Profil</b>.</li>
    <li>Tippen Sie auf den **Hinzufügen +** Button neben dem Text **HBA-Karte**.</li>
    <li>Tippen Sie auf **Verifizierung starten**.</li>
    <li>Geben Sie Ihre CAN ein und tippen Sie auf **Bestätigen**. Sie können die 6 Ziffern lange Zahl Ihrer HBA Karte entnehmen.</li>
    <li>Geben Sie Ihre HBA PIN ein und tippen Sie auf **Bestätigen**.</li>
    <li>Platzieren Sie nun den Chip Ihrer HBA Karte auf dem NFC Lesegerät Ihres Mobilgerätes. Bewegen Sie die Karte ein wenig, bis die App **Karte wird gelesen...** anzeigt. Dann halten Sie die Karte still, bis sie erfolgreich registriert wurde.</li>
    <li>Falls die Karte nicht gelesen werden konnte, tippen Sie auf **Erneut scannen** und platzieren Sie die Karde wieder auf dem NFC Lesegerät. Wenn die Karte korrekt platziert ist, bewegen Sie diese nicht mehr, bis sie erfolgreich registriert wurde. Möglicherweise kann auch das Entfernen der Handyhülle den Scan erleichtern.</li>
  </ol>
  </TabItem>
</Tabs>
