## Hinzufügen von Benutzern zur Freigabeliste

Falls Sie beide Teil der selben Organisation sind oder Sie beide bereits eine [eHBA-Karte registriert haben](./use-hba), müssen Sie sich nicht gegenseitig zur Freigabeliste hinzufügen. Dann können Sie einfach [diesen Anweisungen](../using-famedly/chats#direktchat-starten) folgen, um einen Chat zu starten.
Falls Sie einen Benutzer kontaktieren möchten, auf den das nicht zutrifft, müssen Sie sich zunächst gegenseitig zur [Freigabeliste](./approval-list) hinzufügen.   

Sie können sich entweder durch Scannen der Kontakt-QR-Codes oder durch manuelle Eingabe der Benutzer-ID gegenseitig hinzufügen.  

### Anderen Benutzern den QR-Code zeigen

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>Klicken Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf **Profil**.</li>
    <li>Klicken Sie auf den **QR-Code Icon Button** neben Ihrem Namen.</li>
    <li>Lassen Sie den anderen Benutzer Ihren QR-Code scannen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf **Profil**.</li> 
    <li>Tippen Sie auf den **QR-Code Icon Button** neben Ihrem Namen.</li>
    <li>Lassen Sie den anderen Benutzer Ihren QR-Code scannen.</li>
  </ol>
  </TabItem>
</Tabs>

### Die eigene Benutzer-ID kopieren

Falls Sie keinen Zugriff auf Mobilgeräte haben oder Benutzer über weite Entfernung hinzufügen müssen, sollten
Sie Ihre Benutzer-ID mit dem anderen Benutzer teilen und ihn bitten, dasselbe zu tun, damit Sie sich gegenseitig
zur Freigabeliste hinzufügen können.  

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>Klicken Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf **Profil**.</li>
    <li>Klicken Sie auf den **QR-Code Icon Button** neben Ihrem Namen.</li>
    <li>Klicken Sie auf den **Kopieren-Button** links von Ihrer Benutzer-ID.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf **Profil**.</li> 
    <li>Tippen Sie auf den **QR-Code Icon Button** neben Ihrem Namen.</li>
    <li>Tippen Sie auf den **Kopieren-Button** links von Ihrer Benutzer-ID.</li>
  </ol>
  </TabItem>
</Tabs>

### Andere Benutzer mittels QR-Code zur Freigabeliste hinzufügen

<Tabs>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf **Sicherheit**.</li>
    <li>Tippen Sie auf **Freigabeliste**.</li>
    <li>Tippen Sie auf den **QR-Code Button** in der unteren rechten Ecke.</li>
    <li>Tippen Sie auf **QR-Code scannen**.</li>
    <li>Scannen Sie den QR-Code mit Ihrer Kamera.</li>
  </ol>
  </TabItem>
</Tabs>

### Andere Benutzer mittels Benutzer-ID zur Freigabeliste hinzufügen

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>Klicken Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf **Sicherheit**.</li>
    <li>Klicken Sie auf **Freigabeliste**.</li>
    <li>Klicken Sie auf den **QR-Code Button** in der unteren rechten Ecke.</li>
    <li>Geben Sie die Benutzer-ID des Benutzers, den Sie zur Freigabeliste hinzufügen möchten, in das Eingabefeld **Kontakt mit Benutzer-ID hinzufügen** ein.</li>
    <li>Klicken Sie auf den **+ Button**.</li>
    <li>Falls der andere Benutzer Sie bereits hinzugefügt hat, klicken Sie auf **Chat starten**, andernfalls können Sie den Button **QR-Code teilen** anklicken, um Ihren QR-Code zu teilen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialen, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf **Sicherheit**.</li>
    <li>Tippen Sie auf **Freigabeliste**.</li>
    <li>Tippen Sie auf den **QR-Code Button** in der unteren rechten Ecke.</li>
    <li>Geben Sie die Benutzer-ID des Benutzers, den Sie zur Freigabeliste hinzufügen möchten, in das Eingabefeld **Kontakt mit Benutzer-ID hinzufügen** ein.</li>
    <li>Tippen Sie auf den **+ Button**.</li>
    <li>Falls der andere Benutzer Sie bereits hinzugefügt hat, tippen Sie auf **Chat starten**, andernfalls können Sie den Button **QR-Code teilen** antippen, um Ihren QR-Code zu teilen.</li>
  </ol>
  </TabItem>
</Tabs>
