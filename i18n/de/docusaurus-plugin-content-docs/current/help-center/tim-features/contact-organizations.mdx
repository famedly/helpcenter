## Benutzer anderer Organisationen anschreiben 
Um Benutzer anderer Organisationen anschreiben zu können, müssen Sie Ihre [eHBA Karte registrieren](./use-hba#ehba-karte-registrieren).  
Außerdem müssen Sie sich vorher gegenseitig zur [Freigabeliste](./approval-list) hinzufügen. 
Falls Sie nun noch keinen Chat gestartet haben, können Sie der Anleitung zum [Direktchat Starten](../using-famedly#direktchat-starten) folgen.  

## Kontaktpunkte anderer Organisationen anschreiben 

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Vergewissern Sie sich, dass Sie im Bereich "Chats" sind.</li>
    <li>Klicken Sie auf die **+ Taste** neben dem Filter im oberen linken Bereich Ihres Bildschirms.</li>
    <li>Klicken Sie auf **Organisationen** in der oberen rechten Ecke des Dialogs.</li>
    <li>Wählen Sie die Organisation, die Sie kontaktieren möchten direkt aus oder suchen Sie nach ihr.</li>
    <li>Klicken Sie auf den Kontaktpunkt, den Sie anschreiben möchten.</li>
    <li>Klicken Sie auf **Nachricht senden**, um den Chat zu starten.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Vergewissern Sie sich, dass Sie im Bereich "Chats" sind.</li>
    <li>Tippen Sie auf die **+ Taste** neben dem Filter im oberen linken Bereich Ihres Bildschirms.</li>
    <li>Tippen Sie auf **Organisationen** in der oberen rechten Ecke des Dialogs.</li>
    <li>Wählen Sie die Organisation, die Sie kontaktieren möchten direkt aus oder suchen Sie nach ihr.</li>
    <li>Tippen Sie auf den Kontaktpunkt, den Sie anschreiben möchten.</li>
    <li>Tippen Sie auf **Nachricht senden**, um den Chat zu starten.</li>
  </ol>
  </TabItem>
</Tabs>
