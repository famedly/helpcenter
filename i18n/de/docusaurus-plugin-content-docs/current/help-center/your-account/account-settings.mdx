## Ein Profilbild hochladen oder bearbeiten

Ein Profilbild zu haben ist nützlich, da es anderen Benutzern ermöglicht, Sie leicht zu identifizieren und zu überprüfen, ob sie mit der richtigen Person kommunizieren. Es fügt dem Gespräch auch eine persönliche Note hinzu und macht es ansprechender.

### Profilbild hinzufügen oder ändern

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf Ihrem Profilbild oder Ihre Initialien, um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Klicken Sie auf "Galerie eröffnen".</li>
    <li>Wählen Sie ein Bild aus Ihren Dateien.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie auf Ihrem Profilbild oder Ihre Initialien, um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Tippen Sie auf "Galerie eröffnen" or "Kamera eröffnen".</li>
    <li>Wählen Sie ein Bild aus Ihren Dateien.</li>
  </ol>
  </TabItem>
</Tabs>

### Profilbild entfernen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>

    <li>Klicken Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Klicken Sie auf "Galerie eröffnen".</li>
    <li>Klicken Sie auf "Profilbild entfernen".</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>

    <li>Tippen Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Tippen Sie auf "Profilbild entfernen" </li>
  </ol>
  </TabItem>
</Tabs>

## Benachrichtigungen

Wir verlassen uns auf die zeitnahe Kommunikation zwischen unseren Nutzern, und Benachrichtigungen stellen sicher, dass Sie über neue Nachrichten oder Updates in Echtzeit informiert werden. Ohne Benachrichtigungen können Sie wichtige Nachrichten verpassen und es können Verzögerungen in Ihrer Kommunikation entstehen.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf Ihr <b>Profilbild oder Namensinitials</b> um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf das Feld <b>Benachrichtigungen</b>.</li>
    <li>Klicken Sie auf den Schalter, um die Benachrichtigungen zu aktivieren oder zu deaktivieren.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Klicken Sie auf Ihr <b>Profilbild oder Namensinitials</b> um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf das Feld <b>Benachrichtigungen</b>.</li>
    <li>Klicken Sie auf den Schalter, um die Benachrichtigungen zu aktivieren oder zu deaktivieren.</li>
  </ol>
  </TabItem>
</Tabs>

## Stil

Das Aktivieren des dunklen Modus in Famedly kann die Augenbelastung verringern und es einfacher machen, länger zu arbeiten. Sie können auch den Stil auf "System Theme" einstellen, um das Theme basierend auf den Systemeinstellungen zu wechseln.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Um die Einstellungen zu öffnen, klicken Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Klicken Sie auf <b>Stil</b>.</li>
    <li>Wählen Sie unter <b>Theme</b> den gewünschten Stil aus.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Um die Einstellungen zu öffnen, klicken Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Tippen Sie auf <b>Stil</b>.</li>
    <li>Wählen Sie unter <b>Theme</b> den gewünschten Stil aus.</li>
  </ol>
  </TabItem>
</Tabs>

## Verzeichnis

Sie können es Personen aus anderen Organisationen erlauben, Ihr Konto zu finden. Das ist sinnvoll, wenn Sie mit Personen außerhalb Ihrer eigenen Organisation kommunizieren möchten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Um die Einstellungen zu öffnen, klicken Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Klicken Sie auf <b>Verzeichnis</b>.</li>
    <li>Aktivieren oder deaktivieren Sie die Option, von Personen außerhalb Ihrer Organisation gefunden zu werden.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Um die Einstellungen zu öffnen, tippen Sie auf Ihr Profilbild oder Ihre Initialien.</li>
    <li>Tippen Sie auf <b>Verzeichnis</b>.</li>
    <li>Aktivieren oder deaktivieren Sie die Option, von Personen außerhalb Ihrer Organisation gefunden zu werden.</li>
  </ol>
  </TabItem>
</Tabs>

## Passwort ändern

Wenn Sie die Möglichkeit sehen, das Passwort in den Einstellungen zu ändern, können Sie das Passwort ändern. Wenn Sie die Möglichkeit nicht sehen, wird das Passwort in Ihrer Organisation in einem anderen System festgelegt.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Geben Sie Ihr aktuelles Passwort ein</li>
    <li>Geben Sie Ihr neues Passwort ein</li>
    <li>Bestätigen Sie die Änderung Ihres Passwort, indem Sie auf <b>Passwort ändern</b> klicken.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Geben Sie Ihr aktuelles Passwort ein</li>
    <li>Geben Sie Ihr neues Passwort ein</li>
    <li>Bestätigen Sie die Änderung Ihres Passwort, indem Sie auf <b>Passwort ändern</b> klicken.</li>
  </ol>
  </TabItem>
</Tabs>

## Verbundene Geräte

Hier finden Sie alle Geräte die mit Ihrem Gerät verbunden sind. Sie können die Geräte sperren und entfernen. Bitte beachten Sie, dass Sie Zugriff auf den Geräten verlieren, wenn Sie diese ausloggen. Sie müssen sich auf diesen Geräten dann neu einloggen.

## Archiv

Im Archiv finden Sie die archivierten Unterhaltungen. Das Laden kann einige Zeit in Anspruch nehmen. Sie können alte Nachrichten hier lesen und das Archiv leeren.

-   Um alte Unterhaltungen zu lesen, klicken Sie auf die Unterhaltung.
-   Um das Archiv zu leeren, klicken Sie bitte auf das Menü in der Ecke rechts oben.

## Chat-Backup

Abhängig davon in welchem Status sich das Chat-Backup befindet, haben Sie zwei Optionen:

-   Sie können das Chat-Backup einrichten.
-   Sie können die Passphrase für den Chat-Backup zurücksetzen.

In beiden Fällen, stellen Sie sicher, dass Sie die erstellte Passphrase sicher aufheben.

## Ausloggen

Sie können sich ausloggen, wenn Sie sich mit einem anderen Account bei Famedly einloggen möchten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Um die Einstellungen zu öffnen, klicken Sie auf Ihr <b>Profilbild</b>.</li>
    <li>Klicken Sie auf das Logout-Symbol über Ihrem <b>Profilbild</b>.</li>
    <li>Bestätigen Sie das Ausloggen, indem Sie auf <b>Ausloggen</b> klicken.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Um die Settings zu öffnen, tippen Sie auf <b>ihr Profilbild oder ihren Namen</b> in dem Header oben auf dem Bildschirm.</li>
    <li>Tippen Sie auf <b>Ausloggen</b>.</li>
  </ol>
  </TabItem>
</Tabs>
