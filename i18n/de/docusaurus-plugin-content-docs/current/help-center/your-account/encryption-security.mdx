## Geräte verifizieren

Die Nachrichten werden für jedes Gerät einzeln verschlüsselt, daher sollten Sie alle Ihre eigenen Geräte überprüfen, um zu beweisen, dass nicht jemand anderes Ihre Anmeldedaten in die Hände bekommen hat.

Wenn Sie die Ende-zu-Ende-Verschlüsselung auf jedem Gerät verwenden möchten, müssen Sie Ihre Geräte verifizieren. Alle Nachrichten werden für jedes Gerät separat verschlüsselt. Daher ist eine Überprüfung erforderlich, um sicherzustellen, dass die richtige Person die Nachricht erhält.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Um die Einstellungen zu öffnen, klicken Sie auf Ihr <b>Profilbild</b>.</li>
    <li>Klicken Sie auf <b>Verbundene Geräte</b>.</li>
    <li>Wählen Sie das Gerät, das Sie verifiziereun möchten und klicken Sie auf <b>Gerät verifizieren</b>.</li>
    <li>Akzeptieren Sie die Anfrage auf dem Gerät, das Sie verifizieren möchten.</li>
    <li>Gleichen Sie die angezeigten Codes ab.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Um die Einstellungen zu öffnen, tippen Sie auf Ihr <b>Profilbild oder Ihren Namen</b> am oberen Bildschirmrand.</li>
    <li>Tippen Sie auf <b>Verbundene Geräte</b>.</li>
    <li>Wählen Sie das Gerät, das Sie verifizieren möchten und tippen Sie auf die drei kleine Punkte daneben.</li>
    <li>Tippen Sie auf <b>Verifizieren</b>.</li>
    <li>Akzeptieren Sie die Anfrage auf dem Gerät, das Sie verifizieren möchten.</li>
    <li>Gleichen Sie die angezeigten Codes ab.</li>
  </ol>
  </TabItem>
</Tabs>

:::info

Auf diese Weise können auch Geräte gesperrt und entfernt werden.

:::

## Überprüfen, ob Ende-zu-Ende-Verschlüsselung aktiviert ist

Es ist immer nützlich zu überprüfen, ob die Ende-zu-Ende-Verschlüsselung für eine Unterhaltung korrekt aktiviert ist.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat, den Sie kontrollieren möchten.</li>
    <li>Klicken Sie auf das i in der Ecke oben rechts im Gruppenchat um die Details zu öffnen.</li>
    <li>Wählen Sie <b>Verschlüsselung</b>.</li>
    <li>Wenn hinter jedem Namen ein geschlossenes, grünes Schloss zu sehen ist, ist die Ende-zu-Ende Verschlüsselung aktiv.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie den Chat, den Sie kontrollieren möchten.</li>
    <li>Tippen Sie auf der Leiste über dem Chat um die Details zu öffnen.</li>
    <li>Wählen Sie <b>Verschlüsselung</b>.</li>
    <li>Wenn hinter jedem Namen ein geschlossenes, grünes Schloss zu sehen ist, ist die Ende-zu-Ende Verschlüsselung aktiv.</li>
  </ol>
  </TabItem>
</Tabs>
