## Rollen hinzufügen

Rollen dienen als Kontaktstellen für Anfragen an andere Abteilungen, Abteilungen oder externe Organisationen (Beispiele: allgemeine Anfragen, Diabetes Konsultationszeiten, stationäre neurologische Zulassungen, Pflegestation B4). Nur Personen, die Mitglieder einer Rolle sind, werden diese eingehenden Anfragen um ihre Rolle erhalten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Räume</b>.</li>
    <li>Klicken Sie auf <b>+ Neue Gruppe erstellen</b>.</li>
    <li>Geben Sie den Gruppennamen ein und klicken Sie auf das <b>Papierflugzeugsymbol</b>.</li>
  </ol>
  </TabItem>
</Tabs>

## Rollen bearbeiten

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Räume</b>.</li>
    <li>Finden Sie die Rolle, die Sie bearbeiten möchten, und wählen Sie <b>Bearbeiten</b>.</li>
  </ol>
  </TabItem>
</Tabs>

### Einstellungen ändern

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Mitglieder</b>.</li>
    <li>Bearbeiten Sie den Namen, geben Sie eine Willkommensnachricht ein und legen Sie die Sichtbarkeit der Rolle fest.</li>
    <li>Wenn Sie fertig sind, klicken Sie auf <b>Speichern</b>.</li>
  </ol>
  </TabItem>
</Tabs>

### Fügen Sie der Gruppe neue Mitglieder hinzu

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Mitglieder</b>.</li>
    <li>Klicken Sie auf <b>+ Mitglied hinzufügen</b>.</li>
    <li>Finden Sie die Person, die Sie hinzufügen möchten, und klicken Sie auf <b>Hinzufügen</b>.</li>
  </ol>
  </TabItem>
</Tabs>

### Mitglieder entfernen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Mitglieder</b>.</li>
    <li>Finden Sie die Person, die Sie entfernen möchten, und klicken Sie auf <b>Entfernen</b>.</li>
  </ol>
  </TabItem>
</Tabs>

## Rollen löschen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Gehen Sie zum Bereich <b>Räume</b>.</li>
    <li>Finden Sie die Person, die Sie entfernen möchten, und klicken Sie auf <b>Entfernen</b>.</li>
  </ol>
  </TabItem>
</Tabs>
