## Geräte verifizieren

Die Nachrichten werden für jedes Gerät einzeln verschlüsselt, daher sollten Sie alle Ihre eigenen Geräte überprüfen, um zu beweisen, dass nicht jemand anderes Ihre Anmeldedaten in die Hände bekommen hat.

Wenn Sie die Ende-zu-Ende-Verschlüsselung auf jedem Gerät verwenden möchten, müssen Sie Ihre Geräte verifizieren. Alle Nachrichten werden für jedes Gerät separat verschlüsselt. Daher ist eine Überprüfung erforderlich, um sicherzustellen, dass die richtige Person die Nachricht erhält.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Tippen Sie rechts neben dem Filterfeld auf Ihr <b>Profilbild oder Namens Initialen</b> um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf <b>Verknüpfte Geräte</b>.</li>
    <li>Wählen Sie das zu verifizierende Gerät und wählen Sie <b>Verifizieren</b>.</li>
    <li>Akzeptieren Sie die Anfrage auf dem zu verifizierenden Gerät.</li>
    <li>Überprüfen Sie, ob der angezeigte Code der gleiche ist.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Klicken Sie auf Ihr <b>Profilbild oder Namensinitials</b> um zu den Einstellungen zu gelangen.</li>
    <li>Klicken Sie auf <b>Verknüpfte Geräte</b>.</li>
    <li>Wählen Sie das zu verifizierende Gerät und wählen Sie <b>Verifizieren</b>.</li>
    <li>Akzeptieren Sie die Anfrage auf dem zu verifizierenden Gerät.</li>
    <li>Überprüfen Sie, ob der angezeigte Code der gleiche ist.</li>
  </ol>
  </TabItem>
</Tabs>

## Überprüfen, ob Ende-zu-Ende-Verschlüsselung aktiviert ist

Es ist immer nützlich zu überprüfen, ob die Ende-zu-Ende-Verschlüsselung für eine Unterhaltung korrekt aktiviert ist.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat, den Sie überprüfen möchten.</li>
    <li>Klicken Sie auf die drei Punkte in der oberen Leiste.</li>
    <li>Wählen Sie <b>Verschlüsselung</b>.</li>
    <li>Wenn hinter jedem Namen ein geschlossenes, grünes Schloss steht, ist die Ende-zu-Ende-Verschlüsselung aktiviert.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie den Chat, den Sie überprüfen möchten.</li>
    <li>Klicken Sie auf die drei Punkte in der oberen Leiste.</li>
    <li>Wählen Sie <b>Verschlüsselung</b>.</li>
    <li>Wenn hinter jedem Namen ein geschlossenes, grünes Schloss steht, ist die Ende-zu-Ende-Verschlüsselung aktiviert.</li>
  </ol>
  </TabItem>
</Tabs>
