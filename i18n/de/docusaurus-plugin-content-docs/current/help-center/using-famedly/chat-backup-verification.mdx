## QR-Verifikation

Im Login-Prozess werden Sie automatisch zur QR-Code Verifikation weitergeleitet, sofern mindestens eines Ihrer Geräte ein Mobilgerät ist.  

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie in der <b>Verifikationsanfrage</b> auf <b>Fortfahren</b>, um die Bestätigungsanfrage anzunehmen.</li>
    <li>Falls das andere Gerät ein Mobilgerät ist, wird Ihr Browser Tab nun einen QR-Code anzeigen. Andernfalls werden Sie gebeten, stattdessen Zahlen zu vergleichen.</li>
    <li>Scannen Sie nun den QR-Code, der in Ihrem Browser angezeigt wird mit Ihrem Mobilgerät. Falls beide Geräte Web-Browser sind, vergleichen Sie bitte die Zahlen und bestätigen Sie diese.</li>
    <li>Falls Sie den QR-Code nicht scannen können, folgen Sie der einfach Anleitung zur [Zahlen-Verifikation](./chat-backup-verification#zahlen-verifikation).</li>
    <li>Nachdem Sie den QR-Code gescannt haben, werden Sie gefragt, ob wirklich Sie selbst den QR-Code gescannt haben. Klicken Sie auf <b>Ja, das war ich</b> um die Überprüfung abzuschließen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Tippen Sie in der <b>Verifikationsanfrage</b> auf <b>Fortfahren</b>, um die Bestätigungsanfrage anzunehmen.</li>
    <li>Eines Ihrer Geräte zeigt jetzt einen QR-Code an, das andere Gerät öffnet einen QR-Codescanner.</li>
    <li>Bitte scannen Sie den QR-Code.</li>
    <li>Falls Sie den QR-Code nicht scannen können, folgen Sie einfach der Anleitung zur [Zahlen-Verifikation](./chat-backup-verification#zahlen-verifikation).</li>
    <li>Nachdem Sie den QR-Code gescannt haben, werden Sie gefragt, ob wirklich Sie selbst den QR-Code gescannt haben. Tippen Sie auf <b>Ja, das war ich</b> um die Überprüfung abzuschließen.</li>
  </ol>
  </TabItem>
</Tabs>

## Zahlen-Verifikation

Falls beide Geräte Desktop Geräte oder Web Browser sind, werden Sie automatisch anstelle der QR-Verifikation zur Zahlen-Verifikation aufgefordert. Vergleichen Sie einfach die Zahlen und geben Sie an, ob die Zahlen übereinstimmen oder nicht.  

### Zahlen-Verifikation auf Mobilgeräten verwenden

Sometimes you might need to use number verification instead of QR-Verification (e.g. if your phones camera is broken), in this case you can also tap <b>Compare numbers instead of QR-Code</b> below the QR-Code scanner on your mobile device. 

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie in der <b>Verifikationsanfrage</b> auf <b>Fortfahren</b>, um die Bestätigungsanfrage anzunehmen.</li>
    <li>Wenn das andere Gerät ein Mobilgerät ist, wird dieses einen QR-Code-Scanner öffnen. Tippen Sie darunter auf <b>Zahlen vergleichen statt QR-Code</b>.</li>
    <li>Vergleichen Sie die Zahlen auf den Geräten und geben Sie auf beiden Geräten an, ob die Zahlen übereinstimmen oder nicht.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Tippen Sie in der <b>Verifikationsanfrage</b> auf <b>Fortfahren</b>, um die Bestätigungsanfrage anzunehmen.</li>
    <li>Eins der Geräte wird nun einen QR-Code anzeigen und das andere Gerät wird einen QR-Code-Scanner öffnen.</li>
    <li>Tippen Sie auf <b>Zahlen vergleichen statt QR-Code</b> unter dem QR-Code-Scanner.</li>
    <li>Vergleichen Sie die Zahlen auf den Geräten und geben Sie auf beiden Geräten an, ob die Zahlen übereinstimmen oder nicht.</li>
  </ol>
  </TabItem>
</Tabs>

## Verifikation ohne weitere Geräte

Falls Sie keine weiteren Geräte eingeloggt haben oder den Zugriff auf Ihre anderen Geräte verloren haben, können Sie Ihr Backup wiederherstellen, indem Sie Ihre Passphrase eingeben. Diese wurde Ihnen beim Aktivieren des Chat-Backups angezeigt und beginnt mit den Buchstaben <b>Es</b>.  

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf <b>Keinen Zugriff auf andere Geräte?</b>.</li>
    <li>Geben Sie Ihre Passphrase in das Eingabefeld ein.</li>
    <li>Klicken Sie auf <b>Sicherung wiederherstellen</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Tippn Sie auf <b>Keinen Zugriff auf andere Geräte?</b>.</li>
    <li>Geben Sie Ihre Passphrase in das Eingabefeld ein.</li>
    <li>Tippen Sie auf <b>Sicherung wiederherstellen</b>.</li>
  </ol>
  </TabItem>
</Tabs>

## Passphrase vergessen?

Falls Sie Ihre Passphrase vergessen haben und keine anderen Geräte zur Verifikation verfügbar sind, können Sie die Passphrase zurücksetzen. 

:::danger
Sie werden den Zugriff auf Ihre bisherigen Nachrichten verlieren, wenn Sie Ihre Passphrase zurücksetzen. Bitte setzen Sie die Passphrase nur dann zurück, wenn Sie sicher sind, dass Sie keinen Zugriff darauf haben. 
Falls Sie ein Gerät besitzen, auf dem Sie das Chat Backup bereits aktiviert hatten, können die Nachrichten, die dort bereits empfangen und entschlüsselt wurden, nach einem erneuten Login auch weiterhin eingesehen werden.
:::

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf <b>Keinen Zugriff auf andere Geräte?</b>.</li>
    <li>Klicken Sie auf <b>Keinen Zugriff auf die Passphrase?</b>.</li>
    <li>Bitte lesen Sie die Warnungen im Dialog sorgfältig und klicken Sie dann auf <b>Account zurücksetzen</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobil">    
  <ol>
    <li>Tippen Sie auf <b>Keinen Zugriff auf andere Geräte?</b>.</li>
    <li>Tippen Sie auf <b>Keinen Zugriff auf die Passphrase?</b>.</li>
    <li>Bitte lesen Sie die Warnungen im Dialog sorgfältig und tippen Sie dann auf <b>Account zurücksetzen</b>.</li>
  </ol>
  </TabItem>
</Tabs>
