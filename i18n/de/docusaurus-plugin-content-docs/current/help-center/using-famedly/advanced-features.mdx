## Allgemein

Alle erweiterten Funktionen sind über Chatbefehle verfügbar. Sie können die Befehle im Textfeld eines Chatraums eingeben und auf "Senden" klicken. Die erweiterten Funktionen müssen pro Kunde aktiviert werden.

### Sichtbarkeit eines Raumes ändern

Der Benutzer muss Administrator eines Raumes sein.

- `/setHistoryVisibility <invited/joined/shared/worldReadable>` -> Dies ändert die Sichtbarkeit der Historie in diesem Raum.

### Nutzer ignorieren

- `/ignore <@mxid>` -> Dies fügt einen Benutzer zur Ignorieren-Liste hinzu.
- `/unignore <@mxid>` -> Dies entfernt den Benutzer von der Ignorieren-Liste.
- `/ignoreList` -> Dieser Befehl sendet die Liste der ignorierten Benutzer in den Notizraum.

### Abrufen gemeldeter Ereignisse

<b>Der Benutzer muss OrgAdmin sein.</b>

- `/getReports <limit>` -> Dieser Befehl ruft die gemeldeten Ereignisse vom Server ab und veröffentlicht sie im Notizraum. (Limit ist optional)

### Benutzer abmelden

<b>Der Benutzer muss OrgAdmin sein.</b>

- `/forceLogout <user-id>` -> Dieser Befehl meldet einen anderen Benutzer ab.

### Serverbenachrichtigung senden

<b>Der Benutzer muss OrgAdmin sein.</b>

- `/serverNotice <message>` -> Sendet eine Serverbenachrichtigung an alle Benutzer.
