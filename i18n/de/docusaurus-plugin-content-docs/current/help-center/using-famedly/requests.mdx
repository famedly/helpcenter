## Start a Request

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Click the <b>Start Chat</b> button in the top left corner of the app.</li>
    <li>Click <b>Start Request</b>.</li>
    <li>Search and select a organization and a contactpoint.</li>
    <li>Type in the title of the request and attachements if necessary.</li>
    <li>Click on <b>Request</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tap the <b>+ New</b> button at the bottom right of your screen and select <b>Start Request</b>.</li>
    <li>Search and select a organization an a contactpoint.</li>
    <li>Type in the title of the request and attachements if necessary.</li>
    <li>Click on <b>Request</b>.</li>
  </ol>
  </TabItem>
</Tabs>
