## Eine Anfrage starten

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicke auf den <b>Chat starten</b> Button in der oberen linken Ecke der App.</li>
    <li>Klicken Sie auf <b>Anfrage starten</b>.</li>
    <li>Suchen und wählen Sie eine Organisation und einen Kontaktpunkt.</li>
    <li>Geben Sie den Titel der Anfrage ein und fügen Sie bei Bedarf Anhänge hinzu.</li>
    <li>Klicken Sie auf <b>Anfrage</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie auf den <b>+ Neue</b> Button unten rechts auf Ihrem Bildschirm und wählen Sie <b>Anfrage starten</b>.</li>
    <li>Suchen und wählen Sie eine Organisation und einen Kontaktpunkt.</li>
    <li>Geben Sie den Titel der Anfrage ein und fügen Sie bei Bedarf Anhänge hinzu.</li>
    <li>Klicken Sie auf <b>Anfrage</b>.</li>
  </ol>
  </TabItem>
</Tabs>

## Anfragen anzeigen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicke auf das <b>Umschlagsymbol</b> in der oberen linken Ecke.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Klicken Sie auf die <b>Anfrage</b> Schaltfläche in der unteren Menüleiste.</li>
    <li>Anfragen, die Sie gestellt haben <b>Meine Anfragen</b> sowie die Anfragen die Sie von anderen erhalten haben <b>Geteilt mit mir</b> werden angezeigt.</li>
  </ol>
  </TabItem>
</Tabs>
