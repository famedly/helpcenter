## Anruf in Privatchats starten

Mit Famedly können Sie Sprachanrufe oder Videoanrufe tätigen, um sich mit jedem in einem privaten Chat zu verbinden. Lesen Sie weiter, um zu erfahren, wie Anrufe funktionieren. 

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie einen privaten Chat und klicken Sie in der Kopfzeile auf das <b>Videosymbol</b> oder das <b>Hörersymbol</b>.</li>
    <li>Ihr Anruf wird sofort gestartet und das von Ihnen angerufene Mitglied wird aufgefordert, an dem Anruf teilzunehmen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie einen privaten Chat und tippen Sie auf die Kopfzeile, um die Details zu öffnen.</li>
    <li>Tippen Sie auf das <b>Hörersymbol</b>.</li>
    <li>Tippen Sie auf das <b>Videosymbol</b> oder das <b>Hörersymbol</b>.</li>
    <li>Ihr Anruf wird sofort gestartet und das von Ihnen angerufene Mitglied wird aufgefordert, an dem Anruf teilzunehmen.</li>
  </ol>
  </TabItem>
</Tabs>

<img alt="Start a call" src={useBaseUrl('/img/chatroom_private_1_calls.PNG.png')} width="40%"/>

:::info

Alternativ können Anrufe auch über die drei kleinen Punkte in der rechten oberen Ecke oder über die Details im Chatraum gestartet werden.

:::

### Aktionen bei einem Anruf durchführen

Sobald Sie einen Anruf begonnen oder beigetreten sind, können Sie hier folgendes tun:

- Stumm oder laut schalten
- Video ein- oder ausschalten
- Bildschirm teilen (nicht verfügbar auf iOS)
- Anruf pausieren
- Bild vergrößern

## Sprachnachrichten aufnehmen

### Kurze Sprachnachrichten aufnehmen

Sie können Sprachnachrichten von Famedly auf ihrem mobilen Gerät senden.

<Tabs>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie einen Chat.</li>
    <li>Sprechen Sie ihre Nachricht, während Sie das <b>Mikrofon-Icon</b> in der unteren rechten Ecke gedrückt halten.</li>
    <li>Sie können die Nachricht absenden, indem Sie die Schaltfläche loslassen oder abbrechen, indem Sie den Finger nach links schieben und danach loslassen.</li>
  </ol>
  </TabItem>
</Tabs>

### Lange Sprachnachrichten aufnehmen

Wenn Sie lange Nachrichten aufnehmen möchten, empfiehlt es sich, das Sperr-Feature zu nutzen. Damit können Sie das Mobilgerät zeitweise loslassen und die Aufnahme sogar pausieren und später mit der Aufnahme fortfahren, solange Sie den Chat geöffnet lassen.

<Tabs>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie einen Chat.</li>
    <li>Drücken Sie das <b>Mikrofon-Icon</b> und beginnen Sie, Ihre Nachricht einzusprechen.</li>
    <li>Auf dem Display erscheint der Text <b>Zum Sperren wischen</b> und in der Mitte ein <b>Schloss</b>-Symbol. Wischen Sie mit Ihrem Finger bis zum <b>Schloss</b>-Symbol und nehmen Sie Ihren Finger vom Display.</li>
    <li>Nun erscheint in der Mitte unten ein <b>Pause</b>-Symbol. Wenn Sie die Nachricht pausieren möchten, drücken Sie darauf. Zum weiter Aufnehmen können Sie auf das dann erscheinende <b>Wiedergabe</b>-Symbol drücken.</li>
    <li>Sie können die Nachricht absenden, indem Sie den blauen <b>Pfeil</b>-Button rechts unten drücken. Zum Abbrechen können Sie auf das <b>Papierkorb</b>-Symbol tippen.</li>
  </ol>
  </TabItem>
</Tabs>

## Sprachnachrichten abspielen

Um Sprachnachrichten wiederzugeben, gehen Sie wie folgt vor:

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat mit der Sprachnachricht.</li>
    <li>Klicken Sie auf den <b>Play-Button</b> auf der linken Seite der Nachricht.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Öffnen Sie den Chat mit der Sprachnachricht.</li>
    <li>Klicken Sie auf den <b>Play-Button</b> auf der linken Seite der Nachricht.</li>
  </ol>
  </TabItem>
</Tabs>
