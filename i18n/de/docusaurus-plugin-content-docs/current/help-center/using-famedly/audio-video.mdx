## Anruf in Ptivatchats starten

Mit Famedly können Sie Sprachanrufe oder Videoanrufe tätigen, um sich mit jedem in einem privaten Chat zu verbinden. Lesen Sie weiter, um zu erfahren, wie Anrufe funktionieren.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffne einen privaten Chat und klicke in der Kopfzeile auf das <b>Videosymbol</b> oder das <b>Hörersymbol</b>.</li>
    <li>Ihr Anruf wird sofort gestartet und das von Ihnen angerufene Mitglied wird aufgefordert, an dem Anruf teilzunehmen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie einen privaten Chat und klicken Sie iauf die Kopfzeile, um die Details zu öffnen.</li>
    <li>Tippen Sie auf das <b>Hörersymbol</b>.</li>
    <li>Tippen Sie auf das <b>Videosymbol</b> oder das <b>Hörersymbol</b>.</li>
    <li>Ihr Anruf wird sofort gestartet und das von Ihnen angerufene Mitglied wird aufgefordert, an dem Anruf teilzunehmen.</li>
  </ol>
  </TabItem>
</Tabs>

### Aktionen bei einem Anruf durchführen

Sobald Sie einen Anruf begonnen oder beigetreten sind, können Sie hier folgendes tun:

- Stumm oder laut schalten
- Video ein- oder ausschalten
- Bildschirm teilen (nicht verfügbar auf iOS)
- Anruf pausieren
- Bild vergrößern

## Audioclips aufnehmen

Sie können Audio-Nachrichten von Famedly auf ihrem mobilen Gerät senden.

<Tabs>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie einen Chat.</li>
    <li>Sprechen Sie ihre Nachricht, während Sie das <b>Mikrofon-Icon</b> in der unteren rechten Ecke gedrückt halten.</li>
    <li>Sie schicken die Nachricht, indem Sie die Schaltfläche loslassen oder die Nachricht abbrechen, indem Sie den Finger nach links schieben und danach loslassen.</li>
  </ol>
  </TabItem>
</Tabs>

## Audio-Clips abspielen

Um Sprachnachrichten wiederzugeben, gehen Sie wie folgt vor:

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat mit der Audionachricht.</li>
    <li>Klicken Sie auf den <b>Play-Button</b> auf der linken Seite der Nachricht.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie den Chat mit der Audionachricht.</li>
    <li>Klicken Sie auf den <b>Play-Button</b> auf der linken Seite der Nachricht.</li>
  </ol>
  </TabItem>
</Tabs>
