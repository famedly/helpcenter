## Direktchat starten

Direkte Chats in Famedly sind mit einer sicheren Ende-zu-Ende-Verschlüsselung verschlüsselt. Wir stellen sicher, dass nur die Teilnehmer im Chatraum die ausgetauschten Nachrichten entschlüsseln und darauf zugreifen können. Diese Art der Verschlüsselung bietet eine sichere Umgebung für vertrauliche Gespräche.

Um einen direkten Chat mit anderen Personen in Famedly zu starten, folgen Sie diesen Schritten:

<Tabs>
  <TabItem value="web" label="Web" default>
  <ol>
    <li>Vergewissern Sie sich, dass Sie im Bereich "Chats" sind.</li>
    <li>Klicken Sie auf die <b>+ Taste</b> neben dem Filter im oberen linken Bereich Ihres Bildschirms.</li>
    <li>Wählen Sie die Person, die Sie einladen möchten, aus oder suchen Sie sie.</li>
    <li>Klicke auf <b>Chat starten</b> um einen Chat zu starten.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Stellen Sie sicher, dass Sie sich im Bereich "Chats" befinden.</li>
    <li>Tippen Sie <b>+ Taste</b> oben rechts auf dem Bildschirm.</li>
    <li>Wählen oder suchen Sie die Person, die Sie zum Chat einladen wollen.</li>
    <li>Tippen Sie auf den <b>Chat starten</b> um einen Chat zu beginnen.</li>
  </ol>
  </TabItem>
</Tabs>

## Gruppenchat starten

<Tabs>
  <TabItem value="web" label="Web" default>
  <ol>
    <li>Stellen Sie sicher, dass Sie im Bereich "Chats" sind.</li>
    <li>Klicken Sie auf die <b>+ Schaltfläche</b> neben dem Filter im oberen linken Bereich Ihres Bildschirms.</li>
    <li>Klicken Sie auf <b>Neue Gruppe</b> in der oberen rechten Ecke.</li>
    <li>Benennen Sie die Gruppe dementsprechend.</li>
    <li>Wählen oder suchen Sie die Person, die Sie zum Gruppenchat einladen möchten.</li>
    <li>Klicke <b>Erstelle</b> um einen Gruppenchat zu erstellen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Stellen Sie sicher, dass Sie im Bereich "Chats" sind.</li>
    <li>Tippen Sie auf <b>+ Taste</b> oben rechts auf dem Bildschirm.</li>
    <li>Tippen Sie auf <b>Neue Gruppe</b> in der oberen rechten Ecke.</li>
    <li>Benennen Sie die Gruppe dementsprechend.</li>
    <li>Wählen oder suchen Sie die Person, die Sie zum Gruppenchat einladen möchten.</li>
    <li>Tippen Sie auf <b>Erstelle</b> um einen Gruppenchat zu erstellen.</li>

  </TabItem>
</Tabs>

Expertentipp: Suchen Sie ein Ende-zu-Ende-verschlüsseltes Notizbuch? Erstellen Sie einfach einen Gruppen-Chat nur mit sich selbst, benennen Sie diesen nach "Notizen" und Sie haben einen sicheren Ort, um Dateien und Text zu speichern.

:::

## Chat beitreten

Um an einem Chat teilzunehmen, müssen Sie von einem Mitglied dieses Chats eingeladen werden.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Sie können nur einem Chat beitreten, wenn Sie eine Einladung erhalten haben.</li>
    <li>Alle Einladungen finden Sie oben in der Chat-Liste.</li>
    <li>Klicken Sie auf den Chat in den Sie eingeladen wurden.</li>
    <li>Einladung annehmen.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
    <ol>
    <li>Sie können nur einem Chat beitreten, wenn Sie eine Einladung erhalten haben.</li>
    <li>Alle Einladungen finden Sie oben in der Chat-Liste.</li>
    <li>Klicken Sie auf den Chat in den Sie eingeladen wurden.</li>
    <li>Die Einladung annehmen.</li>
  </ol>
  </TabItem>
</Tabs>

:::danger

Sollten Sie die Einladung ablehnen, müssen Sie die einladenden Person bitten, Sie erneut einzuladen.

:::

## Chat stummschalten

Es ist möglich, Benachrichtigungen für Chats zu stummschalten. Dies kann hilfreich sein, wenn viele Mitglieder in einer Gruppe kommunizieren und Sie nicht unbedingt über neue Nachrichten informiert werden möchten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf die <b>drei Punkte</b> in der oberen rechten Ecke des Chats, den Sie stummen möchten.</li>
    <li>Klicken Sie auf den <b>Chat stummschalten</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie auf die <b>drei Punkte</b> in der oberen rechten Ecke des Chats, den Sie stummschalten möchten.</li>
    <li>Tippe auf den <b>Chat stummschalten</b>.</li>
  </ol>
  </TabItem>
</Tabs>

:::tip

Sie können die Stummschaltung des Gruppenchats wieder aufheben, indem Sie die gleichen Schritte ausführen und auf <b>Stummschaltung aufheben</b> auswählen.

:::

## Chat verlassen

Sie können Chaträume jederzeit verlassen und damit das Gespräch beenden. Wenn Sie einen Chat verlassen haben, ist der Verlauf des Chats immer noch im Archiv verfügbar.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf die <b>drei Punkte</b> in der oberen rechten Ecke eines Chats.</li>
    <li>Klicken Sie auf <b>Chat verlassen</b>.</li>
    <li>Klicken Sie auf <b>Chat verlassen</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippe auf die <b>drei Punkte</b> in der oberen rechten Ecke des Chat-Bildschirms.</li>
    <li>Tippe auf <b>Chat verlassen</b>.</li>
    <li>Tippe auf <b>Chat verlassen</b>.</li>
  </ol>
  </TabItem>
</Tabs>

:::note

Der Chat wurde für Sie archiviert. Sie können das Archiv in den Einstellungen finden und somit ältere Inhalte nachlesen.

:::

* * *

## Verlassene Chats löschen

Um Speicherplatz auf Ihrem Gerät zu sparen, können Sie das Archiv leeren und alle archivierten Chats löschen. Um einen Chat auf dem Gerät löschen zu können, müssen Sie ihn zuerst verlassen.

:::note

Es ist derzeit nicht möglich ,als Administrator einen Chat für alle Mitglieder zu löschen.

:::

:::note

Das Laden des Archivs kann einige Zeit in Anspruch nehmen.

:::

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf Ihr <b>Profilbild</b> oder <b>Namensinitials</b> in der oberen linken Ecke des Bildschirms, um die Einstellungen zu öffnen.</li>
    <li>Klicken Sie auf <b>Archiv</b>.</li>
    <li>Klicken Sie auf die <b>drei Punkte</b> in der oberen rechten Ecke des Bildschirms.</li>
    <li>Klicken Sie auf <b>Archiv leeren</b>.</li>
    <li>Klicken Sie auf <b>Löschen</b></li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tippen Sie auf Ihr <b>Profilbild</b> oder <b>Name Initiale</b> oben links um die Einstellungen zu öffnen.</li>
    <li>Tippen Sie auf <b>Archiv</b>.</li>
    <li>Tippe auf die <b>drei Punkte</b> in der oberen rechten Ecke des Bildschirms.</li>
    <li>Tippen Sie auf <b>Archiv leeren</b>.</li>
    <li>Tippe auf <b>Lösche</b></li>
  </ol>
  </TabItem>
</Tabs>

## Gruppenchat Einstellungen

### Gruppenname bearbeiten

Der Raumname erscheint in der Gruppenüberschrift und Administratoren der Gruppe können diesen ändern. Es gibt keine Beschränkung der Zeichenlänge und es können Emoji verwendet werden. Wir empfehlen jedoch, den Gruppenbetreff kurz zu halten. Kann bis zu 250 Zeichen lang sein und kann Emoji enthalten.

### Add people to a group

Groups are the best way to keep conversations about various projects, topics or teams organised in Famedly. Add people to groups to bring the right people and information together in one place. You can add an unlimited number of people to a group. The larger the groups become, the more confusing the collaboration can become. We therefore recommend consciously considering which people are added.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Click on the <b>three dots</b> in the top right corner of a group's screen to open the group details.</li>
    <li>Click on <b>Members</b>.</li>
    <li>Click <b>＋</b> in the upper right corner.</li>
    <li>Click on the person you want to add.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tap <b>⠇</b> in the top right corner of a group's header.</li>
    <li>Select the person you want to add.</li>
  </ol>
  </TabItem>
</Tabs>

### Remove people from a group

Famedly organises conversations between several people in groups. Groups provide order and clarity for work. Create topic-based groups or groups for teams to encourage collaboration. If a person no longer needs to be part of a group, the person can be removed. Administrators of a group can remove people from the group.

If you remove a person from a group, that person can no longer participate in the communication, but still has access to old messages within the group.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Click on the <b>three dots</b> in the top right corner of a group's screen to open the group details.</li>
    <li>Click on <b>Members</b>.</li>
    <li>Click on <b>Remove User</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tap on the header of a group to open the group details.</li>
    <li>Tap on <b>Participants</b>.</li>
    <li><b>Tap and hold</b> on the person you want to remove.</li>
    <li>Tap <b>Remove Person Name</b>.</li>
  </ol>
  </TabItem>
</Tabs>

<img alt="Add group members" src={useBaseUrl('/img/chatroom_group_details_participant_settings.PNG.png')} width="40%"/>

:::danger

You cannot remove the person if they are also an administrator. In this case, you must ask the person to leave the group on their own.

:::

### Restrict messaging to group admins

Groups can be set so that only administrators can send messages. This can be particularly useful for groups with many people in which announcements are to be shared. In this case, only admins of a group can send announcements.

## Externes Label

Um sicherzustellen, dass keine Informationen unbeabsichtigt an externe Organisationen weitergegeben werden, haben wir ein "externes" Label eingeführt. Wenn Sie Personen zu einem Chatraum einladen oder Chaträume mit externen Personen bereits bestehen, werden diese mit dem Label "Extern" gekennzeichnet.

## Tipps und Best Practices

-   Vergewissern Sie sich, dass die von Ihnen erstellten oder betreuten Gruppenchats ein klares Thema und eine klare Beschreibung haben, damit die Leute verstehen, wofür sie verwendet werden.
-   Individualisieren Sie Ihre Gruppenchats, indem Sie ein Profilbild für Ihren Gruppenchat erstellen
-   Archivieren Sie Gruppenchats, wenn sie nicht mehr benötigt werden.
