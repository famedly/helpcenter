## Dateien in Famedly teilen


<Tabs> <TabItem value="desktop" label="Desktop" default>
1. Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie die Datei teilen möchten.
2. Klicken Sie auf das **Büroklammer-Symbol**.
3. Wählen Sie die Datei aus, die Sie freigeben möchten und bestätigen Sie.</TabItem> <TabItem value="mobile" label="Mobile">
1. Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie die Datei teilen möchten.
2. Klicken Sie auf das **Büroklammer-Symbol**.
3. Wählen Sie **Dokument**.
4. Wählen Sie die Datei aus, die Sie freigeben möchten und bestätigen Sie.</TabItem> </Tabs>

## Bilder und Videos teilen


<Tabs> <TabItem value="desktop" label="Desktop" default>
1. Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie ein Bild oder Vide teilen möchten.
2. Klicken Sie auf das **Büroklammer-Symbol**.
3. Wählen Sie das Bild/Video aus Ihrem Dateisystem aus.
4. Klicken Sie auf **Ja**, um das Teilen zu bestätigen..</TabItem> <TabItem value="mobile" label="Mobile">
1. Öffnen Sie eine Gruppe oder einen Privatchat, mit dem Sie ein Bild oder Vide teilen möchten.
2. Tippen Sie **Video**, **Galerie** oder **Kamera**.
3. Wählen Sie ein Bild/Video oder machen Sie ein Bild/Video mit der Kamera.
4. Optional: Tippen Sie auf **Schwarz** wenn Sie Teile des Bildes schwärzen möchten.
5. Tippen Sie auf das **Häkchensymbol**.</TabItem> </Tabs>

## Dateien und Bilder herunterladen

Sie können Dateien und Bilder, die Sie in einem Chat erhalten haben, herunterladen.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Öffnen Sie den Chat mit der Datei, die Sie herunterladen möchten.</li>
    <li>Klicken Sie auf die Datei, dann auf den Downloadpfeil in der oberen rechten Ecke.</li>
    <li>Bestätigen Sie die Benachrichtigung.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Öffnen Sie den Chat mit der Datei, die Sie herunterladen möchten.</li>
    <li>Klicken Sie auf die Datei, dann auf den Downloadpfeil in der oberen rechten Ecke.</li>
    <li>Bestätigen Sie die Benachrichtigung.</li>
  </ol>
  </TabItem>
</Tabs>

## Bildabschnitte unkenntlich machen


<Tabs> <TabItem value="mobile" label="Mobile">
1. Folgen Sie den Schritten zum Teilen eines Bildes.
2. Bevor Sie auf das **Häkchensymbol** tippen, können Sie die Farben schwarz oder weiß auswählen, indem Sie auf das kleine Maler Scroll-Symbol in der unteren rechten Ecke der App klicken, wenn Sie ein Bild ausgewählt haben.
3. Tippen Sie auf das **Häkchensymbol**.</TabItem> </Tabs>
