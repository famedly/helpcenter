## Nachrichten senden

<Tabs>
  <TabItem value="desktop" label="Desktop" default>

## Neue Nachrichten lesen

A <b>bold</b> group chat or private chat in your chats list means that there are unread messages in the conversation. Klicken oder tippen Sie auf den fetten Konversationsnamen, um die ungelesenen Nachrichten anzuzeigen.

Weil sich gelegentlich Fehler einschleichen, ist es in Famedly möglich, Nachrichten zu bearbeiten und zu löschen.

* * *

## Nachrichten bearbeiten

Standardmäßig können alle Mitglieder ihre Nachrichten bearbeiten.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Bewegen Sie den Cursor über die Nachricht, die Sie bearbeiten möchten.</li>
    <li>Klicken Sie auf das ✎ <b>Bleistift-Symbol</b>.</li>
    <li>Bearbeiten Sie Ihre Nachricht im Textfeld.</li>
    <li>Klicken Sie auf das →<b>Senden Symbol</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobil" label="Mobil">
  <ol>
    <li>Tippen Sie auf die Nachricht, die Sie bearbeiten möchten.</li>
    <li>Tippen Sie <b>Bearbeiten</b> um Änderungen vorzunehmen.</li>
    <li>Bearbeiten Sie Ihre Nachricht im Textfeld.</li>
    <li>Tippen Sie auf das ✔️ <b>Häkchen Symbol</b> um den Prozess abzuschließen.</li>
  </ol>
  </TabItem>
</Tabs>

## Nachrichten löschen

Standardmäßig können alle Mitglieder ihre Nachrichten löschen.

:::info

Das Löschen einer Nachricht ist permanent, also überlegen Sie es sich vorher gut.

:::

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Click on the message you want to delete.</li>
    <li>Click the <b>Delete</b>.</li>
    <li>Click <b>Yes</b> to confirm.</li>
  </ol>
    </TabItem>
    <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tap the message you want to delete.</li>
    <li>Tap <b>Delete</b> to delete the message.</li>
    <li>Tap <b>Yes</b> to confirm.</li>
  </ol>
  </TabItem>
</Tabs>

## Emojis zu Ihren Nachrichten hinzufügen

Emojis sind eine Bilddarstellung von den textbasierten Emoticons. In Famedly sind Emojis sowohl unterhaltsam als auch praktisch für effizienteres Arbeiten: Verwenden Sie Emojis, um ihre Nachrichten zu verfeinern oder fügen Sie Emoji-Reaktionen einer Nachricht hinzu, um schnell darauf zu reagieren.

<Tabs>
  <TabItem value="macos" label="MacOS" default>
  Benutzen Sie die Emoji-Tastatur um ein Emoji zu versenden. Bitte schauen Sie sich den Link unten an, um zu sehen, wie Sie die Emoji-Tastatur öffnen können.

  [Mehr Informationen](https://support.apple.com/de-de/guide/mac-help/mchlp1560/mac#:~:text=Press%20Control-Command-Space.,and%20choose%20Show%20Emoji%20%26%20Symbols.&text=-E%20(if%20the%20Globe%20key,is%20available%20on%20the%20keyboard).)

</TabItem>
  <TabItem value="windows" label="Windows">
  <ol>
    <li>Um in Windows ein Emoji hinzuzufügen, tippen Sie auf das <code>Windows-Logo</code> + <code>.</code> um die Emoji-Tastatur zu öffnen.</li>
  </ol>
    </TabItem>
    <TabItem value="ios" label="iOS">
  <ol>
    <li>Tippen Sie in der Eingabeleiste um die Tastatur zu öffnen.</li>
    <li>Tippen Sie auf das <b>Emoji</b> oder den <b>Globus</b> Knopf auf der Tastatur.</li>
    <li>Tippen Sie auf den gewünschten Emoji um ihn zur Nachricht hinzuzufügen.</li>
  </ol>
    </TabItem>
    <TabItem value="android" label="Android">
  <ol>
    <li>Benutzen Sie die Emoji Tastatur von Ihrem Gerät um ein Emoji einzufügen.</li>
  </ol>
  </TabItem>
</Tabs>

## Mit einem Emoji auf eine Nachricht reagieren

Oftmals kann eine weitere Nachricht durch eine Emoji-Reaktion ersetzt werden. Sie können sehen wer mit einem Emoji reagiert hat, indem Sie mit der Maus oder mit dem Finger die Emoji-Reaktion gedrückt halten.

### Reaktionen hinzufügen

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Klicken Sie auf die Nachricht, worauf Sie reagieren wollen.</li>
    <li>Wählen Sie einen Emoji aus den Vorschlägen oder klicken Sie auf das <b>＋ Plus-Symbol</b> um ein anderes Emoji auszuwählen.</li>
  </ol>
    </TabItem>
    <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Tippen Sie auf die Nachricht, worauf Sie reagieren wollen.</li>
    <li>Wählen Sie einen Emoji aus den Vorschlägen oder tippen Sie auf das <b>＋ Plus-Symbol</b> um ein anderes Emoji auszuwählen.</li>
  </ol>
  </TabItem>
</Tabs>

### Reaktionen entfernen

Klicken oder tippen Sie einfach auf die Reaktion unter der Nachricht, um sie wieder zu entfernen.

:::info

Sie können nur Ihre eigenen Reaktionen löschen.

:::

## Jemanden in einer Nachricht erwähnen

Indem Sie mit dem @ jemanden in einer Nachricht erwähnen, können Sie Personen direkt über etwas benachrichtigen, dass ihre Aufmerksamkeit in Famedly erfordert. Erfahren Sie, wie Sie Personen in Ihren Nachrichten erwähnen können.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Schreiben Sie das <b>@-Symbol</b>, wenn Sie eine Nachricht schreiben.</li>
    <li>Geben Sie den Namen eines Persons ein oder wählen Sie eine Person aus der Liste. Sie können diesen Schritt wiederholen für jede Person, die Sie in einer Nachricht erwähnen wollen.</li>
    <li>Verschicken Sie die Nachricht.</li>
  </ol>
    </TabItem>
    <TabItem value="mobile" label="Mobil">
  <ol>
    <li>Schreiben Sie das <b>@-Symbol</b> in die Nachrichtenleiste.</li>
    <li>Geben Sie den Namen von der Person, die Sie erwähnen wollen ein oder wählen Sie sie aus der Liste aus. Sie können diesen Schritt für jede Person wiederholen, die Sie in Ihrer Nachricht erwähnen möchten.</li>
    <li>Senden Sie Ihre Nachricht.</li>
  </ol>
  </TabItem>
</Tabs>

:::info

Sie können nur Personen erwähnen die Mitglieder der Unterhaltung sind, sowohl in private Unterhaltungen als auch in Gruppenchats. Wenn Sie eine Person erwähnen wollen, die nicht Mitglied der Gruppe ist, muss diese Person zuerst der Gruppe hinzugefügt werden.

:::

## Add to Notes

You can save messages and media from other conversations by adding it to <b>Notes</b>.

<Tabs>
  <TabItem value="desktop" label="Desktop" default>
  <ol>
    <li>Click on the message you want to save.</li>
    <li>Now, choose <b>Add to Notes</b> from the list.</li>
    <li>The message will be saved in a private chat named <b>Notes</b>.</li>
    <li>You can click on the <b>Go to Notes</b> button that appears at the bottom of the window (which will disappear after a few seconds), to check.</li>
  </ol>
    </TabItem>
    <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Tap the message you want to save.</li>
    <li>Now, choose <b>Add to Notes</b> from the list.</li>
    <li>The message will be saved in a private chat named <b>Notes</b>.</li>
    <li>You can click on the <b>Go to Notes</b> button that appears at the bottom of the screen (which will disappear after a few seconds), to check.</li>
  </ol>
  </TabItem>
</Tabs>

:::info

The **Notes** chat will be created and become visible only after saving your first message with the **Add to Notes** action.

:::

## Nachrichten formatieren

Mit Hilfe der Formatierung können Sie Ihre Nachrichten in Famedly detaillierter und klarer gestalten. Sie können Markup-Sprache verwenden und den Text von ausgewählten Sonderzeichen umgeben.

Wenn Sie Famedly auf Ihrem Desktop oder mobil verwenden, können Sie Nachrichten mit diesen Befehlen formatieren.

| Format                | Beschreibung                                                                        |
| --------------------- | ----------------------------------------------------------------------------------- |
| Fett                  | Text mit Sternen umgeben: **Ihr Text**                                              |
| Kursiv                | Text mit Unterstrichen umgeben: _Ihr Text_                                          |
| Durchgestrichen       | Text mit Tilden umgeben: ~Ihr Text~                                                 |
| Code                  | Text mit Backticks umgeben: `Ihr Text`                                              |
| Zitat                 | Fügen Sie eine spitze Klammer vor den Text: >Ihr Text                               |
| Code block            | Fügen Sie drei Backticks vor und nach dem Text hinzu: `Ihr Text`                    |
| Überschrift 1         | Fügen Sie eine Route ein und drücken Sie die Leertaste vor dem Text: # Ihr Text     |
| Überschrift 2         | Fügen Sie zwei Routen ein und drücken Sie die Leertaste vor dem Text: ## Ihr Text   |
| Überschrift 3         | Fügen Sie drei Routen ein und drücken Sie die Leertaste vor dem Text: ### Ihr Text  |
| 1.  Nummerierte Liste | Schreiben Sie 1 und einen Punkt und dann ein Leerzeichen vor dem Text: 1. Ihr Text. |
| -   Aufzählungsliste  | Fügen Sie ein Sternchen und ein Leerzeichen vor Ihrem Text ein.                     |
