# Willkommen zu unseren API Docs

Die Verbindung von Matrix und FHIR bietet mehrere Vorteile, da Matrux eine offene, dezentralisierte Kommunikationsplattform ist und FHIR (Fast Healthcare Interoperability Resources) ist ein Standard für das Austauschen von Gesundheitsdaten. Durch die Integration beider Technologien können Vorgänge effektiv über APIs umgesetzt werden.

Auf diesen Seiten stellen wir Ihnen unsere Konzepte vor.
