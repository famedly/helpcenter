# Willkommen zu unseren API Docs

The connection between Matrix and FHIR can provide numerous benefits, given that Matrix is an open, decentralized communication platform, and FHIR (Fast Healthcare Interoperability Resources) 
is a standard for exchanging healthcare data. By integrating both technologies, actions can be effectively implemented through APIs.

On these pages, we will introduce you to the concepts we are pursuing.
