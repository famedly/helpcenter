## Open the Context Menu for Message Functions

:::info
Editing messages will only be available if you are not a TI-Messenger user since editing messages is not allowed on TI-Messengers.
:::

### The Quick Menu (Desktop Devices Only)

On desktop devices, you can access quick options by hovering your mouse cursor over a message.  

There you will find the following options: 

1. A quick selection for üëç and üëé emoji reactions.
2. A button that opens an emoji selection for reactions.
3. An arrow left button to reply to the message.
4. A pen button to edit the message (only for your own text messages).
5. A button to copy the message (only for text messages).
6. A download button (only for files such as videos, documents and audio content including voice messages).
7. A **‚†á** button that opens the full message context menu.

### The Full Context Menu

In the message context menu, you will find the following options: 

1. **Read**: Open a list of read receipts from other users.
2. **Copy**: Copy the entire content of the message.
3. **Add to Notes**: Forwards the message to your Notes. If you have not saved any notes yet, **Notes** will now appear pinned at the top of your chat list.
4. **Reply**: This allows you to write a reply to a message. The message you are replying to will be partially displayed above your message. When you tap on it, it will scroll up to the original message.
5. **Edit** (only for your own text messages): When you choose to edit and send the message with changes, the text *edited* will appear in the bottom right.
6. **Forward**: This allows you to forward a message to another chat.
7. **Delete** (only for your own messages): Delete a message; it will then be displayed as a deleted message without the original content.
8. **Report offensive content** (only for messages from other users): You can report offensive content.

  <ol>

## Format your messages

Formatting helps add detail and clarity to your messages in Famedly. You can use markup language and surround text with special characters.

When using Famedly on your desktop or mobile, you can format your messages with these markup commands.

| Format                | Description                                                                         |
| --------------------- | ----------------------------------------------------------------------------------- |
| **Bold**              | Surround text with asterisks:¬†`**your text**`                                       |
| _Italicize_           | Surround text with underscores: `_your text_`                                       |
| ~Strikethrough~       | Surround text with tildes: `~your text~`                                            |
| `Code`                | Surround text with backticks: `` `your text` ``                                     |
| Block quote           | Add an angled bracket in front of text: `>your text`                                |
| Code block            | Add three backticks in front and after of text: ` ```your text``` `                 |
| Header 1              | Add `#` then press Space in front of text                                           |
| Header 2              | Add `##` then press Space in front of text                                          |
| Header 3              | Add `###` then press Space in front of text                                         |
| 1. Ordered list       | Add 1 and a full stop, then press¬†Space¬†in front of text: `1. Your text`            |
| - Bulleted list       | Add an asterisk, then press¬†Space¬†in front of text: `* Your text`                   |
