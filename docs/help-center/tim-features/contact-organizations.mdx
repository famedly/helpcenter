## Contact users from other organizations

To be able to contact users from other organizations, you will need to [register your eHBA](./use-hba#register-ehba-card).  
Also you need to add each other to [approval list](./approval-list) first.  
If you did not start a chat by then, you can follow the steps to [start a direct chat](../using-famedly/chats#start-direct-chat).
