## Add external user to approval list

If both of you are part of the same organization or both of you have [registered an eHBA card](./use-ehba), you will not have to add each other to the approval list. You can follow [these instructions](../using-famedly/chats#start-direct-chat) to start a chat.
If you want to contact a user to whom this does not apply, you must first add each other to the [approval list](./approval-list).  

You can either add each other by scanning the contact QR code or enter the other user's ID manually.   

### Show QR code to other users

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>To access the settings, click on your profile picture or name initials.</li>
    <li>Click on **Profile**.</li>
    <li>Click on the QR code button next to your name.</li>
    <li>Let the other user scan your QR code.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>To access the settings, tap on your profile picture or name initials.</li>
    <li>Tap on **Profile**.</li>
    <li>Tap the QR code button next to your name.</li>
    <li>Let the other user scan your QR code.</li>
  </ol>
  </TabItem>
</Tabs>

### Copy own user ID

If you don't have access to mobile devices or need to add people over distance, you need to share your user ID
and ask other users to send their user IDs to be able to add each other to the approval list. 

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>To access the settings, click on your profile picture or name initials.</li>
    <li>Click on **Profile**.</li>
    <li>Click on the QR code button next to your name.</li>
    <li>Click on the **copy button** next to your user ID.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>To access the settings, tap on your profile picture or name initials.</li>
    <li>Tap on **Profile**.</li>
    <li>Tap the QR code button next to your name.</li>
    <li>Tap the **copy button** next to your user ID.</li>
  </ol>
  </TabItem>
</Tabs>

### Add users to approval list with QR code

<Tabs>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>To access the settings, tap on your profile picture or name initials.</li>
    <li>Tap on **Security**.</li>
    <li>Tap on **Approval list**.</li>
    <li>Tap the QR code button in the bottom right corner.</li>
    <li>Tap on **Scan QR code**.</li>
    <li>Scan the QR code with your camera.</li>
  </ol>
  </TabItem>
</Tabs>

### Add users to approval list with user ID

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>To access the settings, click on your profile picture or name initials.</li>
    <li>Click on **Security**.</li>
    <li>Click on **Approval list**.</li>
    <li>Click on the QR code button in the bottom right corner of the dialog.</li>
    <li>Type in or insert the user ID of the user you want to add to the input field **Add contact via user ID**.</li>
    <li>Click on the **+ button**.</li>
    <li>If the other user already added you, click on **Start chat**, if not you can click on **Share QR code** to share the QR code.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>To access the settings, tap on your profile picture or name initials.</li>
    <li>Tap on **Security**.</li>
    <li>Tap on **Approval list**.</li>
    <li>Tap the QR code button in the bottom right corner.</li>
    <li>Type in or insert the user ID of the user you want to add to the input field **Add contact via user ID**.</li>
    <li>Tap the **+ button**.</li>
    <li>If the other user already added you, tap **Start chat**, if not you can tap **Share QR code** to share the QR code.</li>
  </ol>
  </TabItem>
</Tabs>
