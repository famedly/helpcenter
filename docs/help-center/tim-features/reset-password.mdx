    <h2 class="hero__title">Reset Password</h2>
    <p class="hero__subtitle">How to reset your password if you have forgotten it or want to change it.</p>
  </div>

:::info
If no new browser window or tab opens for login, your account is not on a TIM server and you need to contact your administrator for resetting a lost password.
:::

<Tabs>
  <TabItem value="desktop" label="Desktop">
  <ol>
    <li>Open Famedly web app.</li>
    <li>Enter your login URL into the corresponding input field.</li>
    <li>Click <b>Next</b>.</li>
    <li>A new tab for entering the login credentials will open.</li>
    <li>Enter your username and click **Continue**.</li>
    <li>Click <b>Reset password</b>.</li>
    <li>Check your email inbox for a mail with the title <b>Reset password</b>. If necessary, also check the spam folder.</li>
    <li>Click the <b>Reset password</b> button in that mail.</li>
    <li>Choose a password that complies with the rules below the <b>New password</b> input field.</li>
    <li>Insert your new password into the <b>New password</b> input field.</li>
    <li>Insert your new password again into the <b>Repeat password</b> input field.</li>
    <li>Click <b>Continue</b>.</li>
  </ol>
  </TabItem>
  <TabItem value="mobile" label="Mobile">
  <ol>
    <li>Open Famedly app.</li>
    <li>Enter your login URL into the corresponding input field.</li>
    <li>Tap <b>Next</b>.</li>
    <li>A new tab for entering the login credentials will open.</li>
    <li>Enter your username and tap **Continue**.</li>
    <li>Tap <b>Reset password</b>.</li>
    <li>Check your email inbox for a mail with the title <b>Reset password</b>. If necessary, also check the spam folder.</li>
    <li>Tap the <b>Reset password</b> button in that mail.</li>
    <li>Choose a password that complies with the rules below the <b>New password</b> input field.</li>
    <li>Insert your new password into the <b>New password</b> input field.</li>
    <li>Insert your new password again into the <b>Repeat password</b> input field.</li>
    <li>Tap <b>Continue</b>.</li>
  </ol>
  </TabItem>
</Tabs>
